## Safety & Best Practices

### Handling Electronics
- **ESD (Static) Safety Guide**: 
  - Touch grounded metal before handling
  - Work on non-conductive surface
  - Avoid rubbing on carpets

### Electrical Safety
- **Power Supply Guidelines**:
  - Always use appropriate voltage (5V)
  - Minimum 3A for Raspberry Pi 4
  - Official supply recommended

- **GPIO Safety**:
  - Never connect mains voltage (110V/220V) to GPIO
  - Use appropriate resistors with LEDs
  - Research before connecting unknown components

### Data Safety
- **Backup Important Files**:
  - SD cards can fail
  - Regular backups to USB or cloud
  - Version control for code (Git)

- Amazon: Mix of official and third-party

### What to Look For
- **Raspberry Pi**: Buy official boards only (avoid clones)
- **Power Supplies**: Official supply or 5V/3A+ certified
- **SD Cards**: SanDisk or Samsung, Class 10 minimum
- **Cables**: Quality matters - cheap HDMI cables often fail

## Vocabulary & Glossary

### Hardware Terms
- **GPIO**: General Purpose Input/Output - pins for connecting electronics
- **SoC**: System on Chip - CPU, GPU, RAM all in one chip
- **Heatsink**: Metal piece that dissipates heat from CPU
- **HAT**: Hardware Attached on Top - add-on boards for Pi
- **PoE**: Power over Ethernet - power via network cable

### Software Terms
- **OS**: Operating System - main software managing computer
- **Linux**: Family of open-source operating systems
- **Debian**: Linux distribution that Raspberry Pi OS is based on
- **Terminal**: Text-based command interface
- **Shell**: Program that interprets commands (usually bash)
- **Repository**: Storage location for software packages

### Computing Concepts
- **CPU**: Central Processing Unit - the "brain"
- **RAM**: Random Access Memory - temporary fast memory
- **Storage**: Permanent memory (SD card)
- **Boot**: Starting up the computer
- **Root**: Administrator account with full permissions
- **Sudo**: "Super User Do" - run command with admin rights
