# Materials List

## Per Student Setup

### Hardware (Week 1-4)
- **Raspberry Pi 4 Model B (4GB RAM recommended)** - $55-75
- **MicroSD Card (32GB minimum, Class 10)** - $10
- **Official Raspberry Pi Power Supply (USB-C, 5V/3A)** - $8
- **Micro-HDMI to HDMI Cable** - $8
- **USB Keyboard & Mouse** - $15 (or use existing)
- **Monitor with HDMI input** - Use existing

**Initial Hardware Total: ~$100-120 per student**

### Additional Components (Weeks 4-7)
- **Breadboard** - $5
- **GPIO Breakout Kit** - $10
- **Jumper Wires (M-M, M-F, F-F)** - $8
- **LEDs (assorted colors, 20 pack)** - $5
- **Resistors (assorted, 560Ω-1kΩ range)** - $8
- **Push Buttons** - $5
- **PIR Motion Sensor** - $5
- **Temperature/Humidity Sensor (DHT11 or DHT22)** - $5-10
- **Ultrasonic Distance Sensor (HC-SR04)** - $5

### Robotics Components (Weeks 6-7)
- **Robot Chassis Kit** (wheels, motors, battery holder) - $15-25
- **L298N Motor Driver** - $5
- **AA Batteries (4-6 pack)** - $8
- **Line Following Sensor Module** - $5

### Optional/Shared Equipment
- **Raspberry Pi Camera Module** - $25 (1-2 for class)
- **Portable Monitor** (for demos) - $100 (1 for instructor)
- **USB Hub (powered)** - $15 (helpful for multiple peripherals)
- **Ethernet cables** (for Week 11 clustering) - $5 each

## Software (All Free)
- **Raspberry Pi OS (formerly Raspbian)** - Free download
- **Raspberry Pi Imager** - Free download
- **Python 3** (pre-installed on RPi OS)
- **Visual Studio Code** or **Thonny** (pre-installed)
- **SQLite** (pre-installed)
- **Flask** (Python library)

## Classroom/Shared Materials
- **Multimeter** (2-3 for class) - $20 each
- **Component Storage Box** - $15
- **Anti-static mats** (optional) - $10 each
- **First Aid Kit** (for minor incidents)
- **Cable Management** (ties, clips)

## Budget Estimates
- **Minimal Setup (Weeks 1-3)**: $100-120 per student
- **Complete Course (All 12 weeks)**: $200-250 per student
- **Classroom Shared Materials**: $100-150 total

## Purchasing Recommendations
- **Raspberry Pi Official Store**: Most reliable for RPi and official accessories
- **Adafruit**: Excellent for sensors and breakout boards with good documentation
- **Amazon/AliExpress**: Budget options for generic components (longer shipping)
- **CanaKit**: All-in-one RPi starter kits (often good value)

## Notes
- Consider buying bulk component kits to reduce per-student costs
- Many components are reusable across multiple courses
- Students can keep their Raspberry Pi and continue learning at home
- Check for educational discounts from suppliers
